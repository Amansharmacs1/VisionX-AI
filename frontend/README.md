# 👓 VisionX-AI  
### AI-Powered Smart Glasses for Visually Impaired Assistance

VisionX-AI is an intelligent wearable assistance system designed to help visually impaired individuals navigate their surroundings using Artificial Intelligence, Computer Vision, and Gesture-Based Interaction.

The project provides:
- Real-time Face Recognition
- Smart Obstacle Detection
- Audio Navigation Assistance
- Gesture-Based Controls
- Direction Guidance
- Accessibility-Focused AI Interaction

Built using Python, OpenCV, MediaPipe, and Face Recognition.
